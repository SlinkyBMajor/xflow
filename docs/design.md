## Overview

The app should have a clean a clean and minimalistic design. It should be easy to use and understand. The overall feel should be modern, spacious and friendly.

That said, this is a tool meant for powerusers, so the UI and UX should not avoid including technical details, and breakdowns.

## Avoid

- Emojis
- Heavy use of gradients

## Prefer

- Mono font for technical details and numbers

## Design Inspiration

- Github actions ui
- Github UI
- High contrast UI

## Icons

Always prefer icons from Lucide over custom icons.
